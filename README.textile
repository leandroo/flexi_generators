h1. Flexi Generators

My Rails generator scripts. Generates controllers, views, layouts and stylesheets with:

* Login layout
* App layout
* Menus with Tab Helper plugin 
* Controllers and views generated with Resource Controller plugin

h2. Included Generators

* flexi_authentication: generates user model with sign up and log in.
* flexi_layout: generates generic layout, stylesheet, and helper files.
* flexi_scaffold: generates a controller and optional model/migration.

h2. Quickstart 

1. Install all dependencies with Rails application template:
@rails appname -m http://github.com/leandroo/flexi_generators/raw/master/flexi_template.rb@
2. @rake gems:install@
3. Open a shell and run @rake db:migrate@ to create your database and setup the prepared defaults.
4. Start your server with @./script/server@
5. Open up "http://localhost:3000":http://localhost:3000

h2. Dependencies

Tabs Helper plugin: @ruby script/plugin install git://github.com/danielvlopes/tabs_helper.git@

Resource Controller plugin: @ruby script/plugin install git://github.com/giraffesoft/resource_controller.git@

Will Paginate plugin/Gem: @ruby script/plugin install git://github.com/mislav/will_paginate.git@ or @gem install will_paginate@

and configure your environment.rb

h2. Usage

<pre>
ruby script/generate flexi_layout
...
ruby script/generate flexi_authentication
...
ruby script/generate flexi_scaffold Post title:string body:text
</pre>
